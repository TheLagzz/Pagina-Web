### Opción 2: Ejecutar localmente
Si deseas descargar el código y ejecutarlo en tu propia computadora, sigue estos pasos:

1. **Clonar el repositorio:**
   Abre tu terminal y ejecuta:
   `git clone https://github.com/TheLagzz/Pagina-Web.git`

2. **Abrir la página:**
   * Navega hasta la carpeta recién descargada (`Pagina-Web`).
   * Haz doble clic en el archivo `index.html`.
   * El archivo se abrirá automáticamente de forma local en tu navegador web predeterminado (Chrome, Edge, Firefox, etc.) sin necesidad de instalar un servidor.
